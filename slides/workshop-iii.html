Workshop Guide

How to solve the Instagram REST api workshop exercise

  1. Either install all tools as described in the lecture or start with a project on your very own web server
  2. Log in or sign up for an Instagram user. Maybe you have to have the mobile app to register for a user
  3. Log in to Instagram developer console
  4. Set up a new client
  5. Use an endpoint that only needs client_id for this exercise. E.g: https://api.instagram.com/v1/media/popular?client_id=yourclientid
  6. Either use XMLHttpRequest in javascript or jQuery.get with jQuery to do the requests
  7. Bower pro tip to install jQuery in project (From project root path): 'bower install jquery --save'
