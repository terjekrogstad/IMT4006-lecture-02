Microdata Global Attributes

Microdata is composed of five global attributes

  • itemscope : Scope of a single item
  • itemprop : For assigning properties to elements
  • itemtype : A URL that specifies the vocabulary
  • itemref : References to related elements
  • itemid : A global unique identifier for the element in semantic context
