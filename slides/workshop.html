Workshop

Relevant tools:

Node.js, NPM (Node Package manager), Bower (Frontend package manager), Grunt (Automation tool), Yeoman (Scaffolding web apps) 

Installation instructions (Install Node.js first; comes with NPM installed)
        
    $ npm install -g bower
    $ npm install -g grunt
    $ npm install -g grunt-cli
    $ npm install -g yo
    $ npm install -g generator-webapp
    $ mkdir myinstagramproject
    $ cd myinstagramproject
    $ yo webapp
    $ grunt serve
