HTTP Request Headers

  • Host: (address of host from URL) User-Agent: (identifies web browser)
  • Accept: (types of data browser accepts) 
    Example:
    
Connection: keep-alive
Cache-Control: max-age=0
Accept: text/html,application/xhtml+xml,application/xml;q=0.9,image/webp,*/*;q=0.8
User-Agent: Mozilla/5.0 (Macintosh; Intel Mac OS X 10_10_0) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/39.0.2171.95 Safari/537.36
Accept-Encoding: gzip, deflate, sdch
Accept-Language: nb-NO,nb;q=0.8,no;q=0.6,nn;q=0.4,en-US;q=0.2,en;q=0.2,da;q=0.2
Cookie: _gat=1; _gat_html5rocks=1; _ga=GA1.2.296276867.1420396772; _dc_gtm_UA-51917484-1=1
